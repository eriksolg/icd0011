Projekti ettevalmistus

1. Looge BitBucket-i konto, kui Teil seda veel pole.

2. Looge uus repositoorium nimega icd0011

3. Jagage see projekt kasutajaga "Märt Kalmo".

   Settings -> User and group access
   
   Lisage "Users" listi kasutaja "Märt Kalmo" ("read" õigusega).

   a) Kui teil on mõni repo juba kasutajaga "Märt Kalmo" jagatud (nt. icd0019),
      siis saate kasutaja leida kasutajanime (mkalmo) järgi.

   b) Kui te pole sellele kasutajaga varem midagi jaganud,
      või olete juurdepääsu hiljem tühistanud, siis ainult 
      kasutajanime järgi õiget kasutajat ei leita. Õige kasutaja 
      lisamiseks peate kirjutama kasutaja e-maili, 
      milleks on mart.kalmo@gmail.com.
      
      E-mailiga kasutaja lisamine nõuab manuaalset kutse aktsepteerimist
      ja seega ei saa repo süsteemile koheselt nähtavaks. Peaksite 
      ootama u. päeva, et järgmiste punktidega jätkata.

      Kui kutse on aksepteeritud, siis peaks kasutajate nimekirjas
      mailiaadress muutuma nimeks "Märt Kalmo".

4. Lisage "hook"

   Repository settings -> Webhooks -> Add webhook

   Täitke järgmised väljad:

     Title: mida iganes
     URL: http://ci.itcollege.ee/ci/icd0011/api/notify

5. Kloonige loodud repo oma arvutisse.
   
   git clone https://bitbucket.org/<teie kasutajanimi>/icd0011
   
6. Avage projekt IntelliJ IDEA-s: File -> Open -> leidke kettalt 
   kloonitud kataloog icd0011

7. Looge juurkataloogi fail nimega info.json ja täitke see alloleva infoga.

  "firstName": "<Teie eesnimi>",
  "lastName": "<Teie perekonnanimi>",
  "passwordHash": "<soovitava salasõna räsi>",
  "formOfStudy": "<õppevormi tähis>",
  "iHaveReadTheRulesOfTheCourse": <true, kui olete lugenud>

  Valitud salasõna abil saate hiljem süsteemist oma tulemusi vaadata.
  Salasõna enamasti avatud teksti kujul ei kirjutata, seega peaksite 
  arvutama valitud salasõnast räsi.
  
  Salasõna räsi saate arvutada näiteks järgmistel lehtedel:
  
  http://www.xorbin.com/tools/sha256-hash-calculator
  või
  https://passwordsgenerator.net/sha256-hash-generator/

  Õppevormi tähised on:
  S - päevaõpe;
  O (o-täht, mitte null) - muud õppevormid.

  Kursuse reeglid ja tähtajad
  statsionaarõpe: https://moodle.taltech.ee/mod/page/view.php?id=73780
  sessioonõpe: https://moodle.taltech.ee/mod/page/view.php?id=73746

  Näiteks:
  
  {
    "firstName": "Märt",
    "lastName": "Kalmo",
    "passwordHash": "7ac6b2736f34fce19d6b804eff5a1f5747ada4eaa22f1d49c01e52ddb7875b47",
    "formOfStudy": "S",
    "iHaveReadTheRulesOfTheCourse": true
  }

8. Looge juurkataloogi fail nimega .gitignore

   Selles failis peaksid võimalike muude kirjete kõrval olema read:
   
   *.class
   *.jar
   *.war
   

9. Commit-ige muudatused

10. Push-ige muudatused BitBucket-isse.

11. Lisage commit-ile tag hw01    

12. Kontrollige, kas saate aadressilt http://ci.itcollege.ee/ci/icd0011 sisse logida.
    Kasutaja: BitBucket-i kasutaja
    Salasõna: Punktis 7 valitud salasõna

   Pärast muudatuste push-imist läheb tõenäoliselt mõni minut aega.

NB! Kui ei õnnestu lehele sisse logida, siis kontrollige üle, kas
    - repo nimeks sai ikka icd0011.
    - Bitbucket.org lehel on näha tag hw01 õige commit-i küljes.
    - Kontrollige, et projekt on näha kasutajale mkalmo.
    - Veenduge, et kasutate sisselogimiseks 12. punktis nimetatud infot.
    - Webhook-ide all on näha ka saadetud päringud. Veenduge, et päring oli edukas.

Kui soovite hiljem salasõna vahetada, siis kirjutage info.json faili uue 
salasõna räsi ja commit-ige muudatused.
